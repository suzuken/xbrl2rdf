XLiP Glossary

Terms Definition
XBase resolver
(class XBaseResolver)
 Provides a means of resolution from a URI reference to an absolute URI defined in XML Base.
XLink data set
(class XLinkDataSet)
 Information that is a route to an XLink data model.
All link declaration information is obtained from an XLink data set.
The XLink processor constructs and obtains this data.
XLink data model
 A data model for handling XLink.
XLink processor
(class XLinkProcessor)
 Provides a function for analyzing XML documents including elements defined in XLink.
Analysis results can be obtained as an XLink data set.
XLink exception
(class XLinkException)
 An exception that occurred during the processing of the XLink processor is reported.
XPointer processor
(class XPointerProcessor)
 Provides a function for analyzing fragment identifiers defined in XPointer.
Location information referred to by XPointer and a node set can be obtained as analysis results.
XPointer exception
(class XPointerException)
 An exception that occurred during the processing of the XPointer processor is reported.
Arc information
(class Arc)
 Arc-type element information defined in XLink or the information for which the simple-type element attributes listed below are unified are modeled for this information.
This information is obtained from link declaration information.
  • arcrole attribute
  • title attribute
  • show attribute
  • actuate attribute
  • from attribute
  • to attribute
Arc information list
(interface ArcList)
 Provides a list of arc information.
Arc information list table
(interface ArcHashtable)
 Provides an arc information list in which strings are registered as keys. The following strings can be keys:
  • Value of the from attribute contained in arc information
  • Value of the label attribute contained in locator information
Address information
(class Address)
 The href attributes unified to locator information are modeled for this information.
This information is obtained from locator information.
Title information
(class Title)
 Title-type elements defined in XLink are modeled for this information.
This information is obtained from link declaration information, locator information, and arc information.
Document information table
(interface DocumentHashtable)
 Returns a document corresponding to the XML document processed by the XLink processor.
The search key is the document URI.
Node set
(interface NodeSet)
 Node sets defined in XPath are modeled for this information. This set is a collection of nodes defined in a DOM.
Node location information
(class XPNode)
 Node-type location information is modeled for this information.
For details, see "5. Data Model" of XPath.
This information is obtained as the results of a method provided by the XPointer processor.
Point location information
(class Point)
 Point Location defined in XPointer is modeled for this information.
This information is obtained as the results of a method provided in the XPointer processor.
Message reporter
(class XLinkMessageReporter)
 Interface that notifies an application of an error if an XLink exception occurs in the XLink processor.
Link declaration information
(class XLinkDeclaration)
 The elements listed below and defined in XLink are modeled for this information.
This information is obtained from an XLink data set.
  • extended-type element
  • simple-type element
Link declaration information list
(interface XLinkDeclarationList)
 A collection of link declaration information.
Link declaration information list table
(interface XLinkDeclarationHashtable)
 Returns a link declaration information list corresponding to the XML document processed by the XLink processor. A search key is a DOM Document object corresponding to an XML document.
Range location information
(class Range)
 Range Location defined in XPointer is modeled for this information.
This information is obtained as the results of a method provided by the XPointer processor.
Location set
(class LocationSet)
 The location set defined in XPointer is modeled for this information. This set is a collection of location information.
Location information
(interface Location)
 Locations defined in XPointer are modeled for this interface.
The interface is installed with node location information, point location information, and range location information.
The interface is obtained as a result of a method provided in the XPointer processor.
Locator information
(class Locator)
 Information of locator-type elements and resource-type elements defined in XLink or the information for which the following attributes of simple-type elements are unified are modeled for this information:
  • href attribute
  • role attribute
  • title attribute
  • label attribute
This information is obtained from link declaration information.
Locator information list
(interface LocatorList)
 A collection of locator information.
Locator information list table
(interface LocatorHashtable)
 Provides a locator information list in which nodes are registered as keys.
