XLiP (XLink Processor) User's Guide

Preface

XLiP (Fujitsu XLink Processor) performs analysis and processing of XLink, which is a link description language for XML documents conforming to specifications drawn up by W3C.
XLip provides a function to manipulate XPointer, which is a language used to point to any location from within an XML document.
The advantages of XLiP are described below.

Advantage

XLink processing by DOM

This processor is the first XLink/XPointer processor designed to operate with DOM, which is the most standard [Steve1]API of XML. Using DOM, users can configure systems flexibly by combining existing software programs and functions.

Supported by Java™

XLiP is supported as a Java™ library. This provides the following advantages:

  • Independence on platforms
    Users can use the functions of XLink/XPointer easily on different kinds of operating systems that support Java™ VM.
  • Capability of linking existing Java™ applications
    Users can develop a wide variety of applications, from applets to servlets, by linking existing Java™ tools.
Support of XLink Ver.1.0 Rec.

This processor supports XLink Ver.1.0 Rec..
