概要

XLinkデータモデルは、XLinkデータセットと、XLinkデータセットと関連を持つ1つ以上のDOMツリーで構成されます。
XLinkデータセットは、XML文書中のXLink記述を解析した結果とDOMツリーとの関係を保持します。
アプリケーションは、このようなXLinkデータモデルの構成により、DOMとXLinkデータセットの連携により、XLinkで記述された情報を自由に参照することができます。

主要クラスの概要

class XLinkDataSet

XLinkデータセットの本体です。
XLinkデータモデルにおいてXLink情報を管理する最上位のクラスで、XLiPによるすべての解析結果は、XLinkデータセットを起点として取得できます。

class XLinkDeclaration

リンク宣言情報です。
XLinkで定義されているextended型要素およびsimple型要素をモデル化したクラスで、配下の要素の情報を取得する起点を提供します。

class Arc

アーク情報です。
XLinkで定義されているarc型要素をモデル化したクラスで、トラバーサルに関連する詳細情報を提供します。

class Locator

ロケータ情報です。
XLinkで定義されているlocator型要素およびresource型要素をモデル化したクラスです。
リンクの開始・終了位置となるリソースの所在情報ならびに関連情報を提供します。

class Address

アドレス情報です。
XLinkで定義されているlocator型要素のhref属性値をモデル化したクラスで、URIと、URIに含まれるフラグメント識別子を提供します。

class Title

タイトル情報です。
title型要素をモデル化したクラスです。
XLinkのグローバル属性として定義されているtitle属性の情報ではありません。
