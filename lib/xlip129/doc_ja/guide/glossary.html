XLiP用語集

用語 定義
XBaseリゾルバ
(class XBaseResolver)
 XML Baseで定義 されている、URI参照から絶対URIへの解決手段を提供します。
XLinkデータセット
(class XLinkDataSet)
 XLinkデータモデルのルートとなる情報です。
すべてのリンク宣言情報は、 XLinkデータセットから取得します。
XLinkプロセッサによって構築し、取得します。
XLinkデータモデル
 XLinkを取り扱うためのデータモデルです。
XLinkプロセッサ
(class XLinkProcessor)
 XLinkで定義されている 要素を含むXML文書を解析する機能を提供します。
解析結果は、XLinkデータセットとして 取得できます。
XLink例外
(class XLinkException)
 XLinkプロセッサの処理中に発生した 例外を通知します。
XPointerプロセッサ
(class XPointerProcessor)
 XPointerで定義されている フラグメント識別子の解析機能を提供します。
解析の結果として、XPointerが指す ロケーション情報 またはノードセットが得られます。
XPointer例外
(class XPointerException)
 XPointerプロセッサの処理中に 発生した例外を通知します。
アーク情報
(class Arc)
 XLinkで定義されている arc型要素の情報、またはsimple型要素の以下の属性を集約した情報を モデル化した情報です。
リンク宣言情報から取得します。
  • arcrole属性
  • title属性
  • show属性
  • actuate属性
  • from属性
  • to属性
アーク情報リスト
(interface ArcList)
 アーク情報のリストを提供します。
アーク情報リストテーブル
(interface ArcHashtable)
 文字列をキーとして登録されたアーク情報リストを 提供します。キーとなる文字列には、以下が用いられています。
  • アーク情報が保持するfrom属性の値
  • ロケータ情報が保持するlabel属性の値
アドレス情報
(class Address)
 ロケータ情報に集約されている href属性をモデル化した情報です。
ロケータ情報から取得します。
タイトル情報
(class Title)
 XLinkで定義されている title型要素をモデル化した情報です。
リンク宣言情報、 ロケータ情報 およびアーク情報から取得します。
ドキュメント情報テーブル
(interface DocumentHashtable)
 XLinkプロセッサが処理を行った XML 文書に対応するDocumentを返します。
検索キーは、文書のURIです。
ノードセット
(interface NodeSet)
 XPathで定義されている、 ノードセットをモデル化した情報で、 DOMで定義されているNodeのコレクションです。
ノードロケーション情報
(class XPNode)
 Node型のロケーション情報をモデル化した 情報です。
詳細については、XPath の"5 Data Model"を参照してください。
XPointerプロセッサが提供するメソッドの結果として取得します。
ポイントロケーション情報
(class Point)
 XPointerで定義されている Point Locationをモデル化した情報です。
XPointerプロセッサが提供するメソッドの結果として取得します。
メッセージレポータ
(class XLinkMessageReporter)
 XLinkプロセッサ内部で XLink例外が発生した際に、 アプリケーションへエラーを通知するためのインタフェースです。
リンク宣言情報
(class XLinkDeclaration)
 XLinkで定義されている 以下の要素をモデル化した情報です。
XLinkデータセットから取得します。
  • extended型要素
  • simple型要素
リンク宣言情報リスト
(interface XLinkDeclarationList)
 リンク宣言情報のコレクションです。
リンク宣言情報リストテーブル
(interface XLinkDeclarationHashtable)
 XLinkプロセッサが処理を行った XML文書に対応するリンク宣言情報リストを返します。 検索キーは、XML文書に対応するDOMのDocumentオブジェクトです。
レンジロケーション情報
(class Range)
 XPointerで 定義されているRange Locationをモデル化した情報です。
XPointerプロセッサが提供するメソッドの結果として取得します。
ロケーションセット
(class LocationSet)
 XPointerで定義 されている、ロケーションセットをモデル化した情報で、ロケーション情報の コレクションです。
ロケーション情報
(interface Location)
 XPointerで 定義されているlocationをモデル化したインタフェースです。
ノードロケーション情報、 ポイントロケーション情報および レンジロケーション情報で実装されています。
XPointerプロセッサが提供するメソッドの結果として取得します。
ロケータ情報
(class Locator)
 XLinkで定義されている locator型要素およびresource型要素の情報、またはsimple型要素の以下の 属性を集約した情報をモデル化した情報です。
  • href属性
  • role属性
  • title属性
  • label属性
リンク宣言情報から取得します。
ロケータ情報リスト
(interface LocatorList)
 ロケータ情報のコレクションです。
ロケータ情報リストテーブル
(interface LocatorHashtable)
 ノードをキーとして登録された ロケータ情報リストを提供します。
