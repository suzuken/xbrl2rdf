XLiP標準仕様一覧

XLiPの動作条件

XLiPは、以下の標準仕様を満たすライブラリを必要とします。
  • The World Wide Web Consortium (W3C) Technical Reports
    • Extensible Markup Language (XML) 1.0 (Second Edition)
    • Namespaces in XML
    • Document Object Model (DOM) Level 2 Core Specification Version 1.0
  • Java Specification Requests
    • Java™ API for XML Processing (JAXP) 1.1

XLiPが準拠する標準仕様

XLiPは、以下の標準仕様に準拠します。
  • W3C勧告
    • XML Linking Language(XLink) Version1.0
    • XML Path Language (XPath) Version 1.0
    • XML Base
  • W3C勧告候補
    • XML Pointer Language (XPointer) Version 1.0
  • その他
    • 上記標準仕様の前提となる各種標準仕様( The Internet Engineering Task Force(IETF) Request for Comments, W3C Technical Reports, ISO等)

制限

XLiPには、以下の制限があります。
  1. フラグメント識別子について
    現在は、xpointerによりDOMツリーの一部(サブツリー)を参照するよう指定された場合でも、サブツリーだけではなく、サブツリーを含むDOMツリー全体を読み込みます。
その他の詳細な制限については、API仕様書を参照してください。
