HOWTO use Jena with an external DIG reasoner

Deprecation of DIG 1.0 interface

With effect from Jena 2.6.0 onwards, we have taken the decision to remove support for the DIG 1.0 external reasoner interface from Jena. DIG 1.0 has not been an actively supported interface specification for some time, and few extant products implement it any more. Moreover, it does not support a complete range of OWL inference operations. Users who wish to continue to interface to an external reasoner while working with updated versions of Jena have two options:

  • (recommended) use the Pellet reasoner, which has a native Jena interface not dependent on DIG 1.0
  • retrieve the DIG 1.0 code from the Jena CVS repository on SourceForge

In the latter case, if there is sufficient demand from the user-base we will provide an easily-downloadable zip or tar archive of the DIG 1.0 sources. If you would benefit from such an archive, please notify the developers via the Jena support list.
